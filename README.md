# Claude Code

## How to use

### Set up marketplace

> [!TIP]
> If you've already setup your marketplace, there are no need to do this everytime.

Firstly, register the marketplace to your Claude Code.

```console
claude plugin marketplace add KeisukeYamashita/claude-code 
```

### Install the plugins

You can run `/plugins` command on Claude code and search the plugins in `Discover` tab or you can run this command:

```console
claude plugin marketplace add <plugin>@KeisukeYamashita/claude-code
```
